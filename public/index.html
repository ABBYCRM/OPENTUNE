OPENTUNE

FOSS, cross-platform, OBD2-compatible ECU tuning suite. Map editor · Datalog · DTC diagnostics · Custom hardware interface.

★ View on GitHub Quick start →

What it does

OBD-II Diagnostics

Read & clear DTCs, pull VIN, walk 22 standard PIDs (RPM, MAP, ECT, STFT, LTFT, MAF, TPS, VBATT, …) on every car since 1996.

Map Editor

2D heatmap table editor for VE fuel, ignition timing, and boost maps. JSON save / export / import. Dirty-cell highlighting.

Live Datalog

Custom-PID sampling, configurable interval, rolling 1000-sample buffer, real-time canvas plot, export to CSV.

OpenTune HW

Build a custom CAN interface (STM32 / ESP32 / RP2040). Spec, JSON protocol, and firmware sketch included.

Transports

Pick the hardware path that fits your build:

  • Simulator — no hardware, plausible data, for dev and CI
  • ELM327 — the classic $15 OBD2 adapter, works for read-only + DTCs
  • J2534 — Tactrix OpenPort 2.0 and clones, full UDS for supported vehicles
  • OpenTune HW — your custom build, lowest latency, boot-mode pin access

Run it

git clone https://github.com/ABBYCRM/OPENTUNE.git
cd OPENTUNE
npm install
npm run dev   # launches Electron + Vite
